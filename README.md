#  Ewallet - Système de Gestion de Portefeuille

Ewallet est une application web moderne permettant de gérer vos transactions financières, de visualiser votre historique et d'effectuer des transferts en toute simplicité.

##  Démo en ligne
Retrouvez la démo interactive ici : [https://hamza-bailla.github.io/Ewallet/](https://hamza-bailla.github.io/Ewallet/)

##  Fonctionnalités
* **Tableau de bord** : Vue d'ensemble de votre solde et de vos activités récentes.
* **Gestion des Transferts** : Effectuez des envois d'argent de manière sécurisée.
* **Historique complet** : Suivez l'ensemble de vos transactions passées.
* **Interface Responsive** : Compatible avec ordinateurs, tablettes et mobiles.

##  Technologies utilisées
* **Frontend** : HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+).
* **Architecture** : Structure MVC (Modèle-Vue-Contrôleur) avec les dossiers `src/Controller` et `src/css`.

##  Structure du projet
* `/assets` : Contient toutes les images et icônes.
* `/src/Controller` : Logique JavaScript de l'application.
* `/src/css` : Feuilles de style pour le design.
* `/` (Racine) : Pages HTML principales (`index.html`, `dashboard.html`, etc.).

##  Installation
Si vous souhaitez tester le projet localement :
1. Clonez le dépôt : `git clone https://github.com/hamza-bailla/Ewallet.git`
2. Ouvrez le dossier dans VS Code.
3. Lancez `index.html` avec l'extension **Live Server**.
